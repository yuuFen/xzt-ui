# xzt-ui

 [![img](https://img.shields.io/badge/License-MIT-yellow.svg)](https://www.npmjs.com/package/LICENSE) 

## Install

`npm install element-ui -S`

## Quick Start

```
import Vue from 'vue'
import Xzt from 'xzt-ui'

Vue.use(Xzt)

// or
import {
Select,
Button
// ...
} from 'xzt-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```
